If you wish to change the item's Sprite, input "Custom" (no quotes) into the Sprite field under the item. 
Then, after exporting your map, unzip it and place an image file for that custom sprite into a folder named "Items".
Example -> 000.png
It is recommended that the image be 16x16 pixels in size.
Below is shown the file directory of an example custom map with custom item sprites.

MyMap.zmap
	-Data.ini
	-Items.ini
	-Items
		-000.png
		-019.png
		(etc.)
